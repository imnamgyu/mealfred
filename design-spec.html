━ 1. PHILOSOPHY ━

디자인 철학

"의사 처방전이 아니라, 옆집 언니의 조언"

학술적으로 정확하되 권위적이지 않게, 따뜻하되 가볍지 않게. 부모가 죄책감 없이 다음 한 끼를 결정할 수 있도록.

3가지 원칙

🧠 1. 신뢰 (Credibility)

학술 출처 항상 명시 (KDRI 2025·WHO·HabEat). 데이터 출처 없이 주장 금지. 게이지·신호등은 결정론적이고 재현 가능.

💛 2. 공감 (Warmth)

"~해야 합니다" → "~해보실래요?". 부모의 시간·죄책감 인지. "1분이면 끝", "한 번이면 OK" 같은 부담 ↓ 카피.

⚡ 3. 즉시성 (Immediacy)

"내일 뭐 줘야 하나" 30초 안에 답. 무한 스크롤·복잡한 메뉴 금지. 핵심 액션은 항상 1-2 탭 거리.

━ 7. MOTION ━

마이크로 인터랙션

  • hover (desktop): transform:translateY(-2px); transition: 0.15s ease
  • tap (mobile): scale(0.97); transition: 0.1s (active 상태)
  • 모달 슬라이드업: translateY(100%) → 0, 0.25s ease
  • 토스트: opacity 0→1, translateY -10→0, 0.2s
  • 스크롤: scroll-behavior:smooth; -webkit-overflow-scrolling:touch
  • 스켈레톤 로딩: gradient shimmer 1.5s loop
  • 이모지 펄스: AI 분석 중 60px 이모지 scale 1↔1.15 1.5s

전환 시간 가이드

  • 마이크로 (hover·tap): 100-150ms
  • 일반 (드롭다운·tooltip): 200ms
  • 패널·모달: 250-300ms
  • 페이지 전환: 300-400ms

━ 9. ACCESSIBILITY ━

접근성 (WCAG AA)

  • 색 대비: 일반 텍스트 4.5:1 이상, 큰 텍스트 3:1 이상
  • 색에만 의존 금지: 신호등에 텍스트 라벨 동반 ("결핍 위험" / "잘 챙김")
  • 터치 영역: 최소 44×44px (iOS HIG 기준)
  • 폰트 크기: 최소 11px (caption), 본문 13px 이상
  • 키보드 네비: tab order 자연스럽게, focus ring 표시
  • 스크린 리더: 이모지에 aria-label, 게이지바에 aria-valuenow
  • 다국어 대비: 영문 fallback 폰트 (system-ui, -apple-system)
  • 저시력 모드: 시스템 dark mode 감지 (prefers-color-scheme) — Phase 2
━ 10. TONE & VOICE ━

톤앤매너 (카피)

10-1. 어조 원칙

  • 지시 X · 권유 O: "~해야 합니다" → "~해보실래요?"
  • 판단 X · 정보 O: "잘못된 식단" → "이번 주 부족한 영양"
  • 전문 용어 → 자연어 번역: "RNI 65% 충족" → "주 3-4회 만남"
  • 긍정 강조: "철 결핍" → "철이 가장 부족 — 채우면 다음 주 A 등급"
  • 1인칭 회피, 친근 2인칭: "당신" 대신 "지우 엄마", "우리 아이"

10-2. Before / After 예시

10-3. 금지 표현

  • ❌ "엄마가" (아빠 사용자 배제) → ✓ "보호자"
  • ❌ "정상 아이" / "비정상" → ✓ "또래 평균"
  • ❌ "조속히" / "반드시" → ✓ "오늘부터", "한 번 시도"
  • ❌ "전문가의 지시에 따라" → ✓ "함께 알아봐요"
  • ❌ "[브랜드명] 이용 시 효과" → ✓ "이렇게 해보면 보통..."
