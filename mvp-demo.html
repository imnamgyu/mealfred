식사 기록 (음식·횟수·반응) "오늘의 미션 — 시도 N번째" 자동 생성. 12~30번 권고(2편 카피)와 자녀 실제 횟수를 매칭해 위로/격려 푸시 → 2편
거부 패턴 (어떤 음식 거부) Food Chaining AI — 거부 음식 → 비슷한 맛/색/모양/식감 4축 변형 추천. Claude Sonnet 4.7 API 호출 → Phase 9 킬러 기능
다양성 점수 (식품군 N종) 다양성 ↓ 알림 — 미시도 식품군 강조. 사회비용(의료비·면역·장 건강) 환기 카피 자동 노출 → 3편
