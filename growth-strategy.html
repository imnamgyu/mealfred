━ 1. MANIFESTO ━

출사표

한국 부모는 매일 묻습니다.
"오늘 우리 아이 뭐 먹여야 할까?"

하지만 검색하면 광고와 맘카페만 나옵니다. 학술 근거? 없습니다. 우리 아이 맞춤? 없습니다. 어린이집 식단에 무엇을 보충해야 할지? 모릅니다.

밀프레드는 이 질문에 매일 데이터로 답합니다. 광고 없이. 구독료 없이. 평생 무료로.

3가지 약속

  1. 도감·블로그·앱은 평생 무료 — 자녀 영양 정보에 매월 결제 요구하지 않습니다.
  2. 키트는 선택, 강요 없음 — 90일 습관이 끝나면 골고루 키트는 무료. 정기 구독은 자발적.
  3. 학술 근거 100% 공개 — KDRI·WHO·HabEat·Cooke·Toomey 등 모든 출처 표기.

왜 우리가 해야 하는가

  • 한국 최초로 KDRI 2025 36 영양소 매핑 · WHO MDD 8 식품군 진단 · 4,432 레시피 풀 구축 완료
  • 1년 무료 기부로 시작하는 B2B = 페인포인트 진심 탐색 (세일즈 X)
  • 밀프레드_Lab 학술 협업 트랙 = 임팩트 + 학술 성취 이중 트랙

✅ 하는 것

━ 4. VIRAL ENGINE ━

바이럴 엔진 — 어떻게 자가 증식하는가

핵심: 광고비 0원. 사용자가 사용자를 데려오고, 사용자가 매출을 만든다. 3개의 강화 루프.

🔁 Loop 1 — SEO 자가 증식 (도감 + UGC)

방문자가 도감 댓글 작성 → UGC가 SEO 점수 ↑ → 더 많은 방문 → 더 많은 댓글

강화 요소: 댓글 작성 시 +500 마일리지 (FR-V12) · 매일 +50 enrich로 페이지 증가

🔁 Loop 2 — 친구 초대 양방향 보상

90일 챌린지 중인 부모 → 같은 고민 가진 부모에게 카톡 공유 → 양방향 적립 → 둘 다 키트 무료 결제 가속

강화 요소: 초대자 +3,000 · 피초대자 +1,000 · 월 5명 한도로 farming 방지

🔁 Loop 3 — 90일 챌린지 완주 후 입소문

완주자가 골고루 키트 무료로 받음 → 후기 작성 시 +5,000 → 인증샷 공유 → 신규 가입자 유입

강화 요소: 키트 받은 순간 "내가 해냈다" 성취감 → 자발적 공유 동기 (Cialdini 일관성 원칙)

왜 결제 차감 마일리지인가? (현금 환전 X)

  • 매출 안전 — 마일리지는 우리 매출만 차감. 슈퍼 사용자도 결국 우리 매출
  • farming 동기 X — 현금 환전 불가 → 부정 적립 동기 자체가 낮음
  • 장부 단순 — 발행·차감만 추적 → 외부 정산 X
  • 인플레 한도 X 가능 — "100만 마일리지 = 키트 10박스 무료" = 우리 입장 어차피 키트 10박스 정기 구매
