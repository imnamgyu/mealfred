부모 수업 트랙 — 35이론 전수조사 커리큘럼 (기본기 21강)

🗺️ 로드맵 · 미구현(설계 정본 v2) 2026-06-11 · 이사님 발제: "1주일 만에 가르치기보다 체득시키는 목적으로 — 학습 판정은 부모의 기록으로, 과제는 점진적으로. 35가지 국제 이론을 전부 뒤져 결론내라." · 조사 범위 = 엔진 정본 이론표 32개 전수(coaching-scenarios §4) + 국제 가이드라인 36종 전수(L_국제가이드라인·지지수 감사) + 방법론 백과사전 10개 문서 전수(00_설계원칙: SOS·Satter·식약처·푸드브릿지·측정검증) · 짝 정본: 주간 케이던스(정본 ③)

결론 한눈에 — 수업 종류 총 21강 = 코어 8(순차) + 선택 8(아이 신호 발동) + 스팟 4(1~2주 단기) + 조건부 1(월령 게이트). 활성 수업은 항상 1개('한 번에 하나'), 주당 실습 2~7회·하루 10분 내. 체득 기간: 기초 체질(코어 1~5) 약 6개월 → 코어 졸업 약 10개월(처방 주간 인터럽트 포함 캘린더 ~1년) → 선택 포함 풀코스 12~18개월. 이 수치는 기존 키트 설계의 "9코스=45주(약 1년)" 로드맵과 독립적으로 수렴함. 32개 이론 중 버려지는 것 0 — 수업 17·엔진 내장 9·지식/스팟 4·임상 경계 2로 전부 소화.

1. 설계 원리 — "가르치기"가 아니라 "체득시키기"

  • ① 수업 = 과제 사다리(2~4단). 각 단은 관측 가능한 부모 행동 1개. 1주 완성이 아니라 단 통과 → 다음 과제로 점진(이사님 지시). 단 통과 기준 = 기록 신호가 주 N회 × 2주 연속(N은 강별 표 참조).
  • ② 졸업(체득) 판정 = '유지 주' 테스트. 마지막 단 통과 후, 편지가 그 주제를 1~2주 쉬어도 기록 신호가 유지되면 졸업 — 채근 없이 유지되는 것이 체화의 정의이며, 이건 매일 기록을 받는 우리만 할 수 있는 측정이다.
  • ③ 페이싱의 과학(왜 속성 불가인가). 노출 반응 곡선 = 4회째 첫 변화 → 5~6회 정체 → 7회째 재증가(de Wild 2021) — 부모가 5회차쯤 포기하는 걸 막는 게 코칭의 핵심 개입 지점. 습관 자동화 중앙값 ~66일(Lally 2009, 18~254일 — 일반 습관과학·신규 인용). 노출 중단 시 6개월 내 거부 재발(de Wild) → 졸업 후에도 유지 레지스트리로 상시 모니터(신호 악화 시 재수강 권고 — 기존 structuredTip·repeatAlert 인프라 재사용).
  • ④ 판정은 전부 부모의 기록으로. 측정검증 정본(12장)의 ◎지표 — 거부율 추이 · 긍/부정 태그 · 시도 식재료 수 · 식재료 등장 빈도 + 구조화 칩(envTablePct·selfPct·texture·meal_time) + 간식 패턴 + 메모 키워드 + 오늘의 질문 답변. 신호가 약한 강은 '오늘의 질문'을 체크 채널로 지정(표에 표기).
  • ⑤ 전달은 기존 아크 5단계 그대로(intro=왜 배우나 → how=방법 한 장면 → obstacle=막힐 때 → observe=어제 기록 짚기 → reinforce=익힘 칭찬). 일간 엔진 무변경 — 수업은 behavior_goal의 두 번째 공급원일 뿐.
  • ⑥ 재수강 캡: 실습 미관측 시 1회 연장, 그래도 안 되면 다음 강(같은 잔소리 3주 금지 — 06-11 사고 교훈). 막힌 강은 유지 레지스트리에 '미이수'로 남겨 한 학기 뒤 재제안.
  • ⑦ 게이트·라이선스 가드: PFD 위험점수 ≥4(또는 ICFQ 적신호 누적)면 수업 보류·전문가 의뢰(12장 산식). SOS 32단계 상세 코드표는 수료자 전용 — 공개된 6대 분류까지만 사용·표기. '치료(Therapy)' 표방 금지. CEBQ·ICFQ 설문 원문 사용 금지(라이선스 보류). 부모에게 '수업·커리큘럼·N강' 단어 비노출(§13 원칙).

7. 기간 계산 — 왜 1년 코스인가

구간구성순수 수업 시간캘린더(처방 인터럽트 +25%)
기초 체질코어 1~5강약 25주약 6개월
코어 졸업코어 1~8강약 42주(평균 5.3주/강)약 10개월~1년
풀코스코어 8 + 선택 평균 2~3강 + 스팟—12~18개월
  • 처방 주간(결핍·레버가 뚜렷한 주)에는 수업이 일시정지하고 처방이 우선 — 그래서 캘린더는 순수 시간보다 ~25% 길다(가정별 편차).
  • 부작위형 강(1·5·7·8)은 졸업 후 유지 모드로 백그라운드 상시 모니터 — 활성 수업은 언제나 1개라 부모 부담 일정.
  • 이 총량은 기존 키트 로드맵 "9코스=45주(약 1년)"(10_노출유지 §7-1)와 독립 계산인데 수렴 — 방법론적 타당성 교차 확인. 구독 12개월 런웨이와도 정합.
  • 속성이 불가한 이유(재강조): 자동화 ~66일 + 노출 8~15회는 격일 리듬으로만 유효(매일 들이밀면 권태 역효과) + 유지 끊으면 6개월 내 재발.
