## 0. 문제 인식 (왜 이 문서?)

기능을 다 넣으려다 **타깃이 흐려졌다.** 영양 점수·36종 신호등·도감 그리드·영양가족 8칸·마일리지·블로그·박스 product… 전부 "종합 식습관 관리 앱"의 기능이지, **편식으로 매 끼니 전쟁 치르는 엄마**를 위한 게 아니다.

→ 다시 한 사람에게 뾰족하게.

## 1. 페르소나 — "편식 전쟁 중인 엄마"

**김지영 (35), 만 3세 딸 엄마**

- **상황**: 아이가 흰밥·김·바나나만 먹음. 채소·고기·생선 거부. 식탁에 새 음식 올리면 운다. 매 끼니가 30분 전쟁.
- **감정**:
  - 😣 **죄책감** — "내가 이유식을 잘못했나, 내 탓인가"
  - 😰 **불안** — "이러다 키 안 크면, 영양 부족하면 어쩌지"
  - 😮‍💨 **무력감** — "뭘 해도 안 먹어. 좋다는 거 다 해봤어"
  - 🫥 **고립** — "맘카페 보면 다른 집 애들은 다 잘 먹는 것 같아. 나만 이런가"
- **공포 트리거**: **"초등학교 급식"** — "남들 앞에서 안 먹으면, 점심 굶으면 어쩌지"
- **현재 행동**: 맘카페 검색, 영양제 사기, 식판·캐릭터 그릇 바꾸기, 유튜브 먹방 틀고 떠먹이기, 몰래 갈아 넣기
- **진짜 니즈 (3가지)**:
  1. **안심** — "당신 탓 아니에요. 정상이에요" (죄책감 해소)
  2. **다음 행동** — "그럼 구체적으로 뭘 어떻게?" (막막함 해소)
  3. **증거** — "조금씩 나아지고 있어요" (무력감 해소)

> 이 엄마에게 "영양 점수 60점"은 또 하나의 죄책감이다. "36종 중 7종 결핍"은 불안만 키운다.
> 필요한 건 **점수가 아니라 손잡아주는 코치**다.

## 2. 골든타임 — "초등 입학 전까지"

### 학술 근거
- **미각 가소성(plasticity)은 영유아기에 최대**. 이 시기 형성된 맛 선호가 성인까지 지속된다.
- **만 6세까지는 편식 유병률·회복률이 모두 높다 → "정상 발달"로 간주**. 이 시기에 다양한 맛에 노출되면 평생 수용 폭이 넓어진다.
- **이후(초등~)에는 식습관이 고착**되어 교정 난이도가 급상승한다.

### 제품 메시지
> **"편식 교정의 골든타임은 초등 입학 전. 지금이 가장 쉬울 때예요."**

- 만 2세~7세(초등 입학 전) = 결정적 시기
- 부모의 공포("초등 급식")를 그대로 동기로 전환: **"초등 가기 전에 잡자"**
- 핵심 행동 원리: **소량 반복 노출 30번** (한 번 거부했다고 끝이 아니다)

출처:
- [Palate Priming in Toddler Years: The Critical Food Window](https://easypeasie.com/blogs/peapod-blog/palate-priming-in-toddler-years-the-critical-food-window)
- [A Narrative Review of Childhood Picky Eating](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6316224/)
- [Decoding Picky Eating in Children](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC12736178/)

## 4. 기능 소거 — 남길 것 / 내릴 것

### ✅ 핵심 (전면) — "안심 → 행동 → 증거" 루프
1. **식사 기록** (거부 추적) — 30초, 메뉴명만
2. **코치 답장** (안심) — "22번 노출 중 8번, 정상이에요" ← 죄책감 해소
3. **친해지기 레시피** (행동) — 거부 식재료를 죽부터 단계별로
4. **노출 진척** (증거) — "30번 향해 가는 중" 시각화

### 🔽 보조로 내림 (헤드라인 X, 깊이 들어가면 제공)
- 영양 점수 / 36종 신호등 → "불안 자극" 대신 **"이만큼 챙기고 있어요" 긍정 프레임**으로만
- 식재료 도감 그리드 → SEO 유입용으로 유지 (앱 내 헤드라인 X)
- 영양가족 8칸 → 진단 안에 부속으로

### ❌ 이 앱에서 뺄 것 (편식 포커스와 무관)
- 마일리지 / 90일 챌린지 게이미피케이션 (M7·M8) — 보류
- 블로그 캐러셀 (홈에서)
- 박스 product (별도 트랙)
- 도감 친해지기 랭킹 전체 페이지 (홈 Top과 중복)
