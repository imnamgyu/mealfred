# 밀프레드 시리즈 A 피치덱 — 디자인 브리프 (Claude 디자인용)

> **용도**: 이 문서 한 장씩을 Claude(또는 디자인 AI)에 그대로 던져 슬라이드를 디자인한다.
> 각 슬라이드는 **목적 / 헤드라인 / 본문·데이터 / 시각화 지시 / 복붙용 프롬프트**로 구성.
> **포맷**: 16:9 가로. 발표용(스크린) + PDF 출력 겸용. 한 장당 핵심 메시지 1개.
> **톤**: 솔직담백. 창업자가 직접 말하듯. 과장·이모지 남발·억지 장치 없이, 사실과 숫자로.
> **출처**: 검증된 실적은 그대로, 추정치는 슬라이드 하단에 작게 "추정"이라고만 표기.

### 타이포
- 폰트: **Pretendard** (없으면 system sans).
- 제목 `h1` clamp(34px,5.4vw,60px), 800, letter-spacing -0.02em.
- 소제목 `h2` clamp(23px,3.7vw,40px), 800.
- 본문 16~20px, line-height 1.6~1.7.
- 큰 숫자(빅메트릭) clamp(34px,6vw,68px), 800, 오렌지.

### 반복 컴포넌트
1. **태그(eyebrow)**: 좌상단, 11~12px, 800, letter-spacing 0.14em, 오렌지 `#C45A00`, 대문자.
2. **페이지 번호**: 우하단, 12px, `#C9B8A8`.
3. **출처 각주**: 추정치·외부 통계는 슬라이드 하단에 10~11px 회색으로 "추정/출처" 한 줄. 그게 전부. (별도 뱃지·정직 박스 없음 — 담백하게.)

### 슬라이드 배경 변주
- `기본`(흰) → `alt`(크림 `#FBF7F2`) → `navy`(네이비 풀블리드, 텍스트 흰) → `dark`(표지/엔딩) 를 리듬감 있게 교차.

# 슬라이드 구성 (총 14장)

순서: 표지 → 문제(3) → 솔루션 과학·AI엔진(1) → **아하모먼트(1)** → SW만으론 부족·키트경제성(1) → 핵심 솔루션=개인화 배송(1) → Why us 제조·물류(1) → 트랙션(1) → 성장엔진·실행전략(1) → 시장(1) → The Ask(1) → 엔딩(1)

> **변경 이력(이번 개정)**: 개리탄식 BET/정직박스 장치 전면 제거(솔직담백). 1~4P 유지 · 5P 수정(AI 코칭 엔진) · 6P 앞 아하모먼트 신설 · 6P 보강(키트 경제성) · 기존 식재료130·푸드체이닝 페이지 삭제 · 그 뒤 전부 개인화 배송/제조능력/트랙션/실행전략으로 재작성.

## S1 — 표지 (dark)

**목적**: 한 문장으로 "이게 뭔지" + "왜 큰지" 각인.

**헤드라인**: 끼니 한 줄로 시작하는 **아이 편식 치료 서비스**
**서브**: "우리 애는 흰밥만 먹어요" — 영유아 4명 중 1명의 현실에서 시작합니다.
**하단 메타칩**: `24년부터 흑자` · `매해 학생 10만 명` · `YoY 100%` · `누적 100만 인분`

**시각화**: 네이비→오렌지 그라데이션 풀블리드. 중앙 큰 타이틀, "편식 치료"만 `#FFB877` 강조. 하단에 칩 4개 가로 배열. 미니멀, 포스터 느낌.

```prompt
16:9 표지 슬라이드. 배경은 #1a2b4a→#243a5e→#C45A00 대각선 그라데이션(풀블리드). 중앙 큰 제목(Pretendard 800, 흰색): "끼니 한 줄로 시작하는" 다음 줄 "아이 편식 치료 서비스" — "편식 치료"만 #FFB877. 제목 아래 1줄 서브(흰 85%): 인용 "우리 애는 흰밥만 먹어요" — 영유아 4명 중 1명의 현실에서 시작합니다. 맨 아래 가로 칩 4개(반투명 흰 보더 pill): "24년부터 흑자", "매해 학생 10만 명", "YoY 100%", "누적 100만 인분". 좌상단 로고 자리. 여백 넉넉, 포스터, 카드 금지.
```

## S2 — 문제 ① 보편성 + 골든타임 (기본)

**목적**: 시장이 작지 않다 + 시간이 정해져 있다(긴급성).

**태그**: PROBLEM
**헤드라인**: 편식은 드문 일이 아니다 — 그리고 **고칠 수 있는 창은 닫힌다**
**본문 포인트**:
- 영유아 **25~46%**가 편식을 겪는다
- 골든타임 **만 2~6세** — 이 시기를 놓치면 식습관이 굳어진다
- 한국 0~6세 **약 230만 명** · 영유아 가구 **약 200만**

**시각화**: 좌측 아이 픽토그램 100개 중 약 35개를 오렌지로(4명 중 1명꼴). 우측 만 2→6세 타임바(2~6세=초록 열림, 이후=회색 닫힘). 230만/200만 큰 숫자.
**각주**: 편식 유병률·인구는 연구·통계청 기반.

```prompt
16:9 슬라이드. 좌상단 eyebrow "PROBLEM"(#C45A00). 제목(네이비 800): "편식은 드문 일이 아니다 — 그리고 고칠 수 있는 창은 닫힌다". 좌측 절반: 아동 픽토그램 10x10 격자, 약 35개 #FF6B1A, 나머지 연회색 #E8DFD4, 캡션 "영유아 25~46%가 편식". 우측 절반: 가로 타임바 만 2→6세, 2~6세 구간 #1B7A3D(열린 창), 6세 이후 회색(닫힘), 라벨 "골든타임: 식습관이 굳기 전 마지막 창". 우하단 큰 숫자 "230만 명 0~6세" "200만 가구". 하단 작은 회색 각주 "연구·통계청 기반". 배경 #FFFDFB. 깔끔한 인포그래픽.
```

## S3 — 문제 ② 막연한 불안 → 잘못된 대응 (alt)

**목적**: 페인의 감정선. 현상유지의 "틀린 해법"을 때린다.

**태그**: PROBLEM
**헤드라인**: 불안한 부모는 **틀린 방법**으로 더 망친다
**본문 포인트(3개 잘못된 대응)**:
- 영양 불안 → **과도한 건기식** (결핍을 추측해서 들이붓기)
- 행동 압박 → 숟가락 들이밀며 **"딱 한 입만 먹자"** + 억지 칭찬 = **최악**
- 정보 과식 → 유튜브·맘카페·소아과 **파편적 조언**, 내 아이엔 안 맞음

**아래 한 줄**: 강요는 거부를 강화한다 (Satter 책임분담 이론)

**시각화**: 좌측 부모가 아이 입에 숟가락 들이미는 라인 일러스트 + "딱 한 입만" 말풍선. 우측 잘못된 대응 3개를 빨강 ✕로, "딱 한 입만"을 가장 크게.

```prompt
16:9 슬라이드, 배경 #FBF7F2. eyebrow "PROBLEM". 제목(네이비 800): "불안한 부모는 틀린 방법으로 더 망친다". 좌측: 미니멀 라인 일러스트 — 부모가 아이 입에 숟가락 들이미는 장면, 말풍선 "딱 한 입만 먹자". 우측: 세로 3가지, 각 앞에 빨강 #C62828 ✕: (1) "영양 불안 → 과도한 건강기능식품" (2) "행동 압박 → '딱 한 입만' + 억지 칭찬 = 최악"(가장 크게 강조) (3) "정보 과식 → 파편적 조언". 맨 아래 회색 캡션 "강요는 거부를 강화한다 — Satter 책임분담 이론". 감정선 있되 절제.
```

## S4 — 문제 ③ 카테고리 주인이 없다 (기본)

**목적**: 큰 빈 시장 + 아무도 풀스택으로 안 함.

**태그**: WHITE SPACE
**헤드라인**: 매일·감정적 고통인데, **카테고리 주인이 없다**
**비교 매트릭스**:
| | 행동·영양 처방 | 아이별 개인화 | 물리 충족 |
|---|---|---|---|
| 밀키트·이유식 | ✕ | ✕ | △(완제품) |
| 영양제 | △(추측) | ✕ | ✕ |
| 육아앱·커뮤니티 | △(정보만) | ✕ | ✕ |
| **밀프레드** | ✅ | ✅ | ✅ |

**아래 한 줄**: 누구도 "끼니 데이터 → 개인화 처방 → 물리 충족"을 한 몸으로 안 한다.

```prompt
16:9 슬라이드. eyebrow "WHITE SPACE". 제목(네이비 800): "매일·감정적 고통인데, 카테고리 주인이 없다". 중앙 4행×3열 비교표: 행=밀키트·이유식/영양제/육아앱·커뮤니티/밀프레드(마지막 행 강조 #FFF4E5), 열=행동·영양 처방/아이별 개인화/물리 충족. 셀 ✕(회색)·△(주황)·✅(그린 #1B7A3D). 밀프레드 행만 전부 ✅+굵게. 표 아래 한 줄(오렌지): "누구도 '끼니 데이터→개인화 처방→물리 충족'을 한 몸으로 하지 않는다." 배경 #FFFDFB.
```

## S5 — 솔루션 ① AI 코칭 엔진 (navy) 〔수정〕

**목적**: 우리 해법이 감(感)이 아니라 **정립된 과학을 하나의 엔진으로 묶은 것**임을 각인.

**태그**: SOLUTION · THE ENGINE
**헤드라인**: 보건복지부 지침과 **35개국 편식 치료 이론을 하나의 AI 코칭 엔진으로**
**본문 포인트**:
- 보건복지부 편식 지침을 비롯해 **전 세계 35개국의 편식 치료 이론·기법**(SOS · Satter DOR · Food Chaining · HabEat · ICFQ · 단순노출 …)을 종합
- 이걸 **하네스(harness)로 묶어 하나의 AI 코칭 엔진으로 직접 개발**
- 논문엔 흩어져 있던 과학을, **제품으로 작동시킨 곳은 우리뿐**

**시각화**: 좌측에서 여러 이론·국가명 라벨들이 화살표로 중앙의 "AI 코칭 엔진" 박스로 수렴(하네스로 묶이는 느낌). 엔진에서 우측으로 "매일 맞춤 코칭" 출력.

```prompt
16:9 슬라이드, 배경 네이비 #1a2b4a(텍스트 흰). 좌상단 eyebrow "SOLUTION · THE ENGINE"(#FFB877). 제목(흰 800, 2줄): "보건복지부 지침과 35개국 편식 치료 이론을 / 하나의 AI 코칭 엔진으로"("AI 코칭 엔진"만 #FFB877).

레이아웃 = 좌→우 깔때기(funnel) 파이프라인 한 흐름. 가운데 빈 공간 없이 캔버스 전체를 3존으로 꽉 채운다.

[존1 — 좌측 약 38%] "35개의 흩어진 과학". 작은 알약(pill) 칩 12~15개를 부채꼴/별자리로 빽빽하게 배치(보건복지부 지침, SOS, Satter DOR, Food Chaining, HabEat, ICFQ, 단순노출, AAP, WHO, KDRI … 나머지는 흐릿한 칩으로 "+28"). 칩들에서 가느다란 선이 전부 오른쪽 한 점으로 모여드는 깔때기 모양. 좌상단에 큰 숫자 "35" + 작게 "개국 지침·이론".

[존2 — 중앙 약 26%] 깔때기가 좁아지며 하나의 발광하는 코어로 수렴 → 중앙 히어로 "AI 코칭 엔진" 칩/캡슐(오렌지 글로우, 화면에서 가장 강조). 코어 위에 "35 → 1" 모티프 작게. 코어 아래 한 줄 "흩어진 이론을 하네스로 묶어 직접 개발". 코어에서 오른쪽으로 굵은 화살표(#FF6B1A) 1개.

[존3 — 우측 약 36%] 화살표가 닿는 곳에 폰 목업 1개 — 그 아이 맞춤 "코치 편지"(따뜻한 문장 2~3줄 예시). 폰 아래 캡션 "엔진이 매일 보내는 코치 편지".

맨 아래 가로 중앙 강조 한 줄(#FFB877): "논문엔 흩어져 있던 과학, 제품으로 작동시킨 곳은 우리뿐." 좌하단 작은 회색 각주 "국내외 편식 치료 지침·이론 종합 · 일부 출처 확인 중". 우하단 페이지번호.

핵심: '많은 것 → 하나로 압축 → 매일 결과물 출력'이 한눈에 읽히게. 태그가 둥둥 떠 보이지 않도록 전부 수렴선으로 엔진에 연결. 여백·죽은 공간 금지. 테크니컬하고 권위 있는 톤.
```

## S6 — 아하모먼트 (alt) 〔신설〕

**목적**: "이게 진짜 된다"를 한 장면으로. 데이터·이론보다 강한, 사람을 설득하는 증거.

**태그**: THE AHA MOMENT
**헤드라인**: 편식이 심한 아이가 **3주 만에 생당근을 스스로 집어 먹었다**
**본문**:
- 편식이 심했던 아이 **3명**을 대상으로 시작
- **3주 만에**, 떠먹여주지 않아도 **스스로** 생당근을 집어 먹음
- **부모님들도 놀란** 순간 — 우리가 찾은 아하 모먼트
