WEEK 1
👀 시각 탐색
친해지기 — 눈으로 탐색
🔄 SOS: Tolerate + Interact 🌉 푸드브릿지 1단계: 간접 노출 5~10% 🎨 감각놀이 · 주 5회
📦 이번 주 재료
🧊 부침두부 1모(150g) 🧊 순두부 1팩(소) 🧊 연두부 1팩(소)
D1
🔍 두부 3종 비교
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부, 순두부, 연두부를 나란히 놓고 비교합니다. 같은 두부인데 형태와 질감이 모두 다르다는 것을 관찰하며 호기심을 키웁니다.
🎯 선생님 핵심 멘트
"오늘은 두부를 먹는 게 아니야! 두부랑 놀 거야~"
"두부가 이렇게 종류가 많아! 딱딱한 두부, 물 같은 두부!"
⚠️ 절대 먹으라고 하지 않기. "눈으로만 구경하는 거야~"
D2
🎨 두부 스탬프 아트
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부를 모양틀로 자른 후 식용색소를 묻혀 도화지에 찍습니다. 네모난 두부 도장이 만들어집니다.
🎯 선생님 핵심 멘트
"두부 도장이 네모야! 다른 모양도 만들어볼까?"
"두부 도장이 톡톡 찍히지?"
⚠️ 선생님이 두부 자르기. 아이는 찍기만 합니다.
D3
💥 두부 해체 탐험
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부를 손으로 으깨며 질감을 탐구합니다. 단단했던 두부가 부서지는 과정을 경험하며 촉감을 느낍니다.
🎯 선생님 핵심 멘트
"두부를 손으로 으깨봐! 어떤 느낌이야?"
"뭉글뭉글 부서진다! 재미있지?"
⚠️ 장갑 착용 선택. 으깬 두부가 바닥에 떨어질 수 있으니 매트 준비.
D4
🏘️ 두부 마을 만들기
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부를 블록처럼 쌓아 두부 마을을 만듭니다. 눈 스티커와 이쑤시개로 캐릭터를 만들어 줍니다.
🎯 선생님 핵심 멘트
"두부를 블록처럼 쌓아보자! 두부 마을이다!"
"두부 캐릭터 완성! 눈알이 귀엽지?"
⚠️ 이쑤시개 사용 시 선생님이 옆에서 반드시 지도합니다.
D5
🎭 색종이 두부 캐릭터
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
흰색, 노란색 색종이로 두부 모양 캐릭터를 만듭니다. 네모난 두부에 표정을 그려 다양한 감정을 표현합니다.
🎯 선생님 핵심 멘트
"색종이로 두부 캐릭터를 만들자!"
"네모 두부 친구! 표정을 그려줄까?"
⚠️ 가위 사용 시 선생님이 보조해 주세요.
WEEK 2
🤚 촉각/후각 탐색
탐색하기 — 손으로 만지기
🔄 SOS: Smell + Touch(초기) 🌉 푸드브릿지 2단계: 간접 노출 → 호기심 🎨 감각놀이 · 주 5회
📦 이번 주 재료
🧊 부침두부 2모 🧊 동결건조두부칩 1봉 🌾 밀가루 50g
D6
🤚 촉감 비교 (부침/순/동결건조)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부(탄탄!), 순두부(미끌!), 동결건조두부칩(바삭!)을 만져봅니다. 같은 두부인데 촉감이 완전히 다르다는 것을 발견합니다.
🎯 선생님 핵심 멘트
"눈 감고 만져봐~ 탄탄한 건 뭘까? 미끄러운 건?"
"같은 두부인데 만지면 느낌이 다 다르지?"
⚠️ 입에 넣지 않도록. "만지는 거야, 먹는 거 아니야~"
D7
👃 냄새 맞추기 (두부/감자/고구마)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
종이컵 3개에 두부, 감자, 고구마를 숨기고 냄새로 두부를 찾는 게임입니다. 두부의 고유한 향을 인지하게 됩니다.
🎯 선생님 핵심 멘트
"코로 킁킁~ 어떤 컵에 두부가 숨어있을까?"
"두부 냄새 맞췄어! 코가 정말 좋구나~"
⚠️ 감자/고구마는 선생님이 미리 준비해 주세요.
D8
🫳 두부 으깨기 놀이 (지퍼백)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부를 지퍼백에 넣고 손으로 으깹니다. 으깨지면서 질감이 변하는 것을 관찰합니다. 으깬 두부로 도화지에 모양을 만들어 봅니다.
🎯 선생님 핵심 멘트
"지퍼백 안에서 두부를 으깨봐! 뭉글뭉글~"
"으깨면 크림 같지? 재미있다!"
⚠️ 지퍼백 밀봉 확인. 터지지 않도록 주의합니다.
D9
🎨 으깬 두부 도화지 아트
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
으깬 두부를 도화지에 손가락으로 펴서 그림을 그립니다. 두부의 하얀색과 부드러운 질감으로 특별한 아트가 완성됩니다.
🎯 선생님 핵심 멘트
"으깬 두부로 그림을 그려볼까? 하얀 크림 같지?"
"두부 그림이다! 부드럽지?"
⚠️ 두부가 바닥에 떨어질 수 있으니 매트 준비.
D10
🫶 두부 반죽 (으깬두부+밀가루)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
으깬 두부에 밀가루를 넣어 부드러운 반죽을 만듭니다. 뭉치고, 부수고, 모양을 만들며 놀이합니다. 두부의 향을 직접 느낍니다.
🎯 선생님 핵심 멘트
"두부랑 밀가루를 섞으면 반죽이 돼!"
"하얀 반죽이다! 두부 냄새가 나지?"
⚠️ 알레르기 확인 (밀가루, 대두) 필수. 장갑 착용.
WEEK 3
👄 입 근처 접근
가까워지기 — 입 근처로
🔄 SOS: Touch(후기) + Taste(초기) 🌉 푸드브릿지 3단계: 소극적 노출 🎨 감각놀이 · 주 5회
📦 이번 주 재료
🧊 동결건조두부칩 1봉 🧊 부침두부 2모 🧊 순두부 1팩 🫕 디핑소스 3종
D11
💥 동결건조 두부칩 부수기
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
동결건조 두부칩을 지퍼백에 넣고 부숩니다. 바삭바삭 부서지는 소리를 즐깁니다. 먹고 싶은 아이는 자연스럽게 먹어도 됩니다.
🎯 선생님 핵심 멘트
"바삭바삭 소리 들려? 더 세게 부숴봐!"
"이게 두부로 만든 과자야~ 신기하지?"
⚠️ 먹고 싶은 아이는 먹어도 OK. 강요는 절대 금지.
D12
⭐ 부침두부 쿠키커터
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부를 1cm 두께로 자르고 쿠키커터로 모양을 만듭니다. 별, 하트 등 다양한 모양의 두부 조각을 접시에 장식합니다.
🎯 선생님 핵심 멘트
"두부를 꾹 눌러서 모양을 만들어볼까?"
"하트 모양 두부! 너무 예쁘다~"
⚠️ 쿠키커터 사용 시 선생님이 보조해 주세요.
D13
😊 두부 접시 얼굴
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
두부 조각으로 종이접시 위에 눈, 코, 입을 만듭니다. 하얀 두부 조각들로 귀여운 캐릭터를 완성합니다.
🎯 선생님 핵심 멘트
"접시 위에 두부로 얼굴을 만들어보자!"
"두부 얼굴이 웃고 있어! 이름을 지어줄까?"
⚠️ 소스는 미니디핑컵에 소량만 제공해 주세요.
D14
👄 두부 립터치 (순두부)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
순두부를 코 끝에 살짝, 볼에 살짝, 입술에 살짝 터치합니다. 안 하고 싶은 아이는 안 해도 됩니다. 순두부의 부드러운 질감에 익숙해집니다.
🎯 선생님 핵심 멘트
"두부가 코 인사해! 볼 인사! 입술 인사!"
"부드럽지? 차가워? 입술에 대면 어떤 느낌이야?"
⚠️ 절대 강요 금지. "하기 싫으면 안 해도 돼~"
D15
🫕 디핑 (간장/참기름소스/케첩)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
부침두부 조각에 간장, 참기름소스, 케첩 3종 소스를 찍어봅니다. 맛보기도 OK, 안 먹어도 OK.
🎯 선생님 핵심 멘트
"어떤 소스에 찍어볼까? 간장? 참기름? 케첩?"
"소스만 맛봐도 좋아! 두부는 안 먹어도 돼~"
⚠️ 먹든 안 먹든 칭찬. "소스에 찍어본 것만으로도 대단해!"
WEEK 4
🍽️ 한 입 도전
조리완료 + 토핑 · 조립
🔄 SOS: Taste(후기) + Eat(초기) 🌉 푸드브릿지 4단계: 직접 노출 90%+ 🍳 조리완료 체험 · 주 3회
📦 이번 주 재료
🍛 마파두부덮밥(조리완료) 🍳 두부계란찜(조리완료) 🍲 된장찌개(조리완료)
D16
🍛 마파두부 덮밥 토핑
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 마파두부덮밥을 전자레인지 2분 데웁니다. 아이가 두부 큐브와 파를 직접 올립니다.
🎯 선생님 핵심 멘트
"덮밥 위에 두부를 올려볼까? 파도!"
"내가 올린 두부야! 한 입 먹어볼까?"
⚠️ 강요 금지. 토핑 올리기만 해도 성공!
D17
🍳 두부 계란찜 토핑
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 두부계란찜을 전자레인지 2분 데웁니다. 아이가 으깬 두부와 깨를 직접 뿌립니다. 부드러운 계란찜과 두부의 조합이 거부감을 줄여줍니다.
🎯 선생님 핵심 멘트
"계란찜 위에 두부랑 깨를 뿌려볼까?"
"부드러운 두부 계란찜! 맛이 어때?"
⚠️ 뿌리기만 해도 OK. 먹는 건 아이가 결정합니다.
D18
🍲 된장찌개 두부 넣기
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 된장찌개를 전자레인지 3분 데웁니다. 아이가 두부 큐브를 직접 찌개에 넣습니다. 두부를 직접 투입하는 행위가 참여감을 높여줍니다.
🎯 선생님 핵심 멘트
"된장찌개에 두부를 넣어볼까? 풍덩!"
"내가 넣은 두부다! 맛있겠다~"
⚠️ 뜨거운 국물 주의! 선생님이 그릇을 잡아주세요.

D19

조리완료 밥과 유부에 두부 다이스를 넣어 유부초밥을 직접 만듭니다. 아이가 재료를 넣고 유부를 접는 재미를 경험합니다.

"내가 만든 유부초밥! 맛있겠다~"
⚠️ 4주차보다 자유롭게. 아이 페이스에 맞춰주세요.
D20
🍚 두부 볶음밥 섞기
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 두부볶음밥을 전자레인지 2분 데웁니다. 아이가 두부 다이스와 김가루를 뿌리고 섞습니다.
🎯 선생님 핵심 멘트
"볶음밥에 두부를 뿌려볼까?"
"내가 만든 두부볶음밥! 두부가 보이지?"
⚠️ 섞기만 해도 칭찬해 주세요.
D21
🍜 두부 잔치국수 + 수료식!
