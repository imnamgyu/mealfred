1 무엇인가

집중 키트 = 도전 식재료 1개를 5주간 집중 공략하는 단발 키트입니다. 아이가 특정 식재료(당근·시금치 등)에서 막혔을 때, 그 하나를 7가지 형태 + SOS Approach 6단계 활동 카드 + 부모 가이드로 단계적으로 노출해 친해지게 합니다. (옛 '편식개선키트'의 진화형)

키트역할구성주기
골고루 키트정기 · AI 개인화 배합분석 기반 극다품종 소량(안 잡힌 식재료)주 1회
집중 키트단발 · 1 식재료 deep dive1 식재료 × 7형태 + SOS 6단계 활동 카드 + 부모 가이드막혔을 때
차별점: 골고루는 넓게(다양성), 집중은 깊게(한 식재료 수용). 둘이 funnel로 연결 — 골고루 쓰다 막히는 식재료가 나오면 그 집중 키트를 권한다.

2 라인업 (5주 코스)

단일 식재료 5주 집중 코스 — 기존 코스 콘텐츠(가이드·코스 페이지) 보유

🥕 당근🧅 양파🟦 두부🍄 버섯🍅 토마토🫑 파프리카🥬 시금치🥦 브로콜리🍆 가지

각 코스 = "왜 이 식재료부터 / 21회의 여정 / 5주간 받을 것 / SOS 단계별 활동". 콘텐츠는 이미 제작돼 있고(코스·가이드 페이지), 제품화·판매·발급만 남음.

3 해야 할 것 (백로그)

기획완료
네이밍·구성 확정 — 골고루/집중 2키트 체계, SOS 6단계, "매주 골고루 막히면 집중"
콘텐츠
5주 코스 콘텐츠 9종 — 코스·가이드 페이지 제작 완료(당근~가지)
설계완료·미구현
⭐ 인플루언서 100% 쿠폰 시스템 — DB influencer_coupons + /admin/coupons(통계·발급폼·내역) + /api/admin/coupons(코드생성·저장·알림톡) + 사이드바 🏷쿠폰. 발급·관리는 바로 구현 가능
외부 의존
쿠폰 알림톡 템플릿 등록 — 카카오 비즈채널 템플릿 심사 4~7일. 승인 전엔 코드 수동 전달, 승인 후 자동 발송(lib/sens.ts에 influencer_coupon 템플릿 추가)
결정필요
판매·결제 채널 — 스마트스토어 vs 앱 내 결제(M10 키트결제). 쿠폰 사용(redeem) 흐름이 여기에 연결
결정필요
가격·원가 — 5주 코스 단가 + 원물 원가(골고루 원물 ≈1,341원 참고). 100% 쿠폰 = 인플루언서 무료 제공분 손익
funnel
골고루 ↔ 집중 연결 — 도감/코칭에서 '막힌 식재료'가 잡히면 그 집중 키트를 추천(앱 내 진입로)

4 인플루언서 쿠폰 — 설계 요약

옛 admin.html을 라이브 어드민으로 이관(집중키트 발급 대상)

  • 발급: 인플루언서 이름·전화·블로그 + 집중키트 코스 선택 → 100% 쿠폰 코드 생성 + (알림톡 발송)
  • 관리: 통계(총 발급·알림톡·사용) + 발급 내역(코드·상태) + 삭제
  • 인증: 옛 비밀번호 방식 → 현 어드민 로그인(@mealfred.com) 게이트로 통일
  • 알림톡: lib/sens.ts 재사용. 쿠폰 템플릿만 신규 등록 필요(위 외부 의존)
한 줄: 쿠폰 발급·코드·관리·인증은 바로 구현, 알림톡 자동발송만 템플릿 승인 대기. 승인 전에도 코드 수동 전달로 운영 가능.
