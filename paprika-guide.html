WEEK 1
👀 시각 탐색
친해지기 — 눈으로 탐색
🔄 SOS: Tolerate + Interact 🌉 푸드브릿지 1단계: 간접 노출 5~10% 🎨 감각놀이 · 주 5회
📦 이번 주 재료
🫑 빨강파프리카 1개 🫑 노랑파프리카 1개 🫑 초록파프리카 1개
D1
🚦 파프리카 신호등 (빨/노/초록)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
빨강, 노랑, 초록 3색 파프리카를 나란히 놓고 신호등처럼 비교합니다. 같은 파프리카인데 색깔이 모두 다르다는 것을 관찰하며 호기심을 키웁니다.
🎯 선생님 핵심 멘트
"오늘은 파프리카를 먹는 게 아니야! 파프리카랑 놀 거야~"
"빨강, 노랑, 초록! 신호등 같지? 어떤 색이 제일 예쁜 것 같아?"
⚠️ 절대 먹으라고 하지 않기. "눈으로만 구경하는 거야~"
D2
🌸 파프리카 단면 스탬프 (꽃모양)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카를 가로로 잘라 단면에 식용색소를 묻혀 도화지에 찍습니다. 꽃 모양 같은 예쁜 패턴이 나타나 아이들이 신기해합니다.
🎯 선생님 핵심 멘트
"파프리카 도장이 꽃 모양이야! 3색으로 찍어볼까?"
"빨강 꽃, 노랑 꽃, 초록 꽃!"
⚠️ 선생님이 파프리카 자르기. 아이는 찍기만 합니다.
D3
🔍 파프리카 해부 탐험
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카를 반으로 잘라 씨앗, 속살, 벽을 관찰합니다. 안이 비어있는 것과 하얀 씨앗이 달려있는 것을 돋보기로 봅니다.
🎯 선생님 핵심 멘트
"파프리카 안이 텅 비어있어! 씨앗은 어디에 있을까?"
"하얀 씨앗을 찾았어! 파프리카 탐험가 대단해~"
⚠️ 돋보기 사용 시 선생님이 옆에서 지도합니다.
D4
🤡 파프리카 캐릭터 만들기
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카의 동글동글한 모양을 활용해 캐릭터를 만듭니다. 눈 스티커와 이쑤시개로 팔다리를 만들어 줍니다.
🎯 선생님 핵심 멘트
"파프리카 얼굴을 만들어볼까? 눈알 붙여!"
"빨강 파프리카 친구, 노랑 친구, 초록 친구!"
⚠️ 이쑤시개 사용 시 선생님이 옆에서 반드시 지도합니다.
D5
🌿 색종이 파프리카 정원
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
빨강, 노랑, 초록 색종이로 파프리카를 접어 도화지에 붙여 정원을 만듭니다. 줄기와 잎도 만들어 꾸밉니다.
🎯 선생님 핵심 멘트
"색종이로 파프리카 정원을 만들자!"
"3색 파프리카 정원이 완성됐어!"
⚠️ 가위 사용 시 선생님이 보조해 주세요.
WEEK 2
🤚 촉각/후각 탐색
탐색하기 — 손으로 만지기
🔄 SOS: Smell + Touch(초기) 🌉 푸드브릿지 2단계: 간접 노출 → 호기심 🎨 감각놀이 · 주 5회
📦 이번 주 재료
🫑 생파프리카 1개 🫑 구운파프리카(진공) 2팩 🫑 동결건조파프리카칩 1봉 🫑 파프리카분말 20g 🌾 밀가루 50g
D6
🤚 촉감 비교 (생/구운/칩)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
생 파프리카(아삭!), 구운 파프리카(부드!), 파프리카칩(바삭!)을 만져봅니다. 같은 파프리카인데 촉감이 완전히 다르다는 것을 발견합니다.
🎯 선생님 핵심 멘트
"눈 감고 만져봐~ 아삭한 건 뭘까? 부드러운 건?"
"같은 파프리카인데 만지면 느낌이 다 다르지?"
⚠️ 입에 넣지 않도록. "만지는 거야, 먹는 거 아니야~"
D7
👃 냄새 맞추기 (파프리카/당근/사과)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
종이컵 3개에 파프리카, 당근, 사과를 숨기고 냄새로 파프리카를 찾는 게임입니다.
🎯 선생님 핵심 멘트
"코로 킁킁~ 어떤 컵에 파프리카가 숨어있을까?"
"파프리카 냄새 맞췄어! 코가 정말 좋구나~"
⚠️ 당근/사과는 선생님이 미리 준비해 주세요.
D8
🍜 파프리카채 놀이 (길게 썰어 국수)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카를 길게 채썰어 국수처럼 놀이합니다. 3색 파프리카 채를 섞어 무지개 국수를 만들어 봅니다.
🎯 선생님 핵심 멘트
"파프리카를 길~게 자르면 국수가 돼! 무지개 국수다!"
"빨강 국수, 노랑 국수, 초록 국수를 섞어볼까?"
⚠️ 채 썰기는 선생님이 합니다. 아이는 놀이만 합니다.
D9
🎨 파프리카 분말 그림
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
풀을 바른 도화지에 파프리카분말을 뿌려 빨간 모래그림을 만듭니다. 분말의 선명한 색과 향이 독특한 작품을 완성합니다.
🎯 선생님 핵심 멘트
"풀 위에 파프리카 가루를 뿌려봐! 빨간 모래 같지?"
"파프리카 향이 나는 그림이다!"
⚠️ 분말이 날릴 수 있으니 천천히 뿌리도록 지도합니다.
D10
🫶 파프리카 반죽 (퓨레+밀가루)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카 퓨레에 밀가루를 넣어 빨간 반죽을 만듭니다. 뭉치고, 부수고, 모양을 만들며 놀이합니다.
🎯 선생님 핵심 멘트
"파프리카 퓨레랑 밀가루를 섞으면 반죽이 돼!"
"빨간 반죽이다! 파프리카 냄새가 나지?"
⚠️ 알레르기 확인 (밀가루) 필수. 장갑 착용.
WEEK 3
👄 입 근처 접근
가까워지기 — 입 근처로
🔄 SOS: Touch(후기) + Taste(초기) 🌉 푸드브릿지 3단계: 소극적 노출 🎨 감각놀이 · 주 5회
📦 이번 주 재료
🫑 동결건조파프리카칩 1봉 🫑 구운파프리카 3팩 🫑 파프리카스틱 10개 🫕 디핑소스 3종
D11
💥 파프리카칩 부수기
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
동결건조 파프리카칩을 지퍼백에 넣고 부숩니다. 바삭바삭 부서지는 소리를 즐깁니다. 먹고 싶은 아이는 자연스럽게 먹어도 됩니다.
🎯 선생님 핵심 멘트
"바삭바삭 소리 들려? 더 세게 부숴봐!"
"파프리카 과자야~ 달콤한 냄새가 나지?"
⚠️ 먹고 싶은 아이는 먹어도 OK. 강요는 절대 금지.
D12
⭐ 파프리카 쿠키커터
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
구운 파프리카를 납작하게 펴서 쿠키커터로 모양을 만듭니다. 다양한 모양의 파프리카 조각을 접시에 장식합니다.
🎯 선생님 핵심 멘트
"파프리카를 꾹 눌러서 모양을 만들어볼까?"
"별 모양 파프리카! 너무 예쁘다~"
⚠️ 쿠키커터 사용 시 선생님이 보조해 주세요.
D13
😊 파프리카채 접시 얼굴
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카 채로 종이접시 위에 눈, 코, 입을 만듭니다. 3색 파프리카채로 화려한 캐릭터를 완성합니다.
🎯 선생님 핵심 멘트
"파프리카 채로 얼굴을 만들어보자! 알록달록~"
"3색 파프리카 얼굴! 이름을 지어줄까?"
⚠️ 소스는 미니디핑컵에 소량만 제공해 주세요.
D14
👄 파프리카 스틱 립터치
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카 스틱을 코 끝에 살짝, 볼에 살짝, 입술에 살짝 터치합니다. 안 하고 싶은 아이는 안 해도 됩니다.
🎯 선생님 핵심 멘트
"파프리카가 코 인사해! 볼 인사! 입술 인사!"
"아삭하지? 냄새 나? 입술에 대면 어떤 느낌이야?"
⚠️ 절대 강요 금지. "하기 싫으면 안 해도 돼~"
D15
🫕 디핑 (허머스/크림치즈/마요)
⏱ 30분 (준비5 + 활동20 + 정리5)
📋 활동 설명
파프리카 스틱에 허머스, 크림치즈, 마요 3종 소스를 찍어봅니다. 맛보기도 OK, 안 먹어도 OK.
🎯 선생님 핵심 멘트
"어떤 소스에 찍어볼까? 허머스? 크림치즈? 마요?"
"소스만 맛봐도 좋아! 파프리카는 안 먹어도 돼~"
⚠️ 먹든 안 먹든 칭찬. "소스에 찍어본 것만으로도 대단해!"
WEEK 4
🍽️ 한 입 도전
조리완료 + 토핑 · 조립
🔄 SOS: Taste(후기) + Eat(초기) 🌉 푸드브릿지 4단계: 직접 노출 90%+ 🍳 조리완료 체험 · 주 3회

D16
🍜 파프리카잡채 토핑
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 파프리카잡채를 전자레인지 2분 데웁니다. 아이가 깨와 파를 직접 올립니다.
🎯 선생님 핵심 멘트
"잡채 위에 토핑을 뿌려볼까?"
"알록달록 파프리카잡채! 맛이 어때?"
⚠️ 강요 금지. 토핑 올리기만 해도 성공!
D17
🍳 파프리카 계란볶음 섞기
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 파프리카계란볶음을 전자레인지 2분 데웁니다. 아이가 숟가락으로 직접 섞습니다.
🎯 선생님 핵심 멘트
"계란이랑 파프리카를 섞어볼까?"
"알록달록한 볶음이다! 한 입 먹어볼까?"
⚠️ 섞기만 해도 OK. 먹는 건 아이가 결정합니다.
D18
🍕 파프리카 미니피자 토핑
⏱ 35분 (준비5 + 활동25 + 정리5)
📋 활동 설명
조리완료 미니피자를 전자레인지 2분 데웁니다. 아이가 파프리카 채와 치즈를 토핑으로 올립니다.
🎯 선생님 핵심 멘트
"피자 위에 파프리카를 올려볼까?"
"내가 만든 파프리카 피자! 맛있겠다~"
⚠️ 올리기만 해도 칭찬해 주세요.
