도감 커뮤니티 구현 완료 — Phase 1+2 & 자동 콘텐츠 엔진

설계(커뮤니티 전략)를 실제 앱으로 빌드했습니다. 마을 탭·도감 §6 노하우·글쓰기·반응·보상·모더레이션 + '밀프레드 코치'가 매일 노하우를 자동 게시하는 콜드스타트 엔진까지.

Architecture · 데이터·동선

둥지(도감) · 광장(마을) · 엔진(자동 콘텐츠)

데이터 모델 (UGC 별도 테이블 — 코퍼스 오염 방지)

  • community_posts — 노하우 글(식재료 배열·본문·사진·방법/성향/난이도·반응 카운트·상태). 작성자 글 + 공식 코치 글(parent_id NULL·is_official).
  • community_reactions — 좋아요/해봤어요/신고(1인 1반응). 트리거가 글 카운트 동기화.
  • point_ledger/point_balance 재사용 — 첫 글·주간·월간 보상은 멱등 RPC로만 지급(append-only 감사).
  • 사진 = Supabase Storage community 버킷(본인 폴더 업로드·공개 읽기).

읽기/쓰기 동선

도감 §6 / 마을→저마찰 글쓰기→매핑엔진 자동 태깅→위험 가드→공개 + 첫글 +500P
다른 엄마→👍/🙌 반응→주간 톱10 정산(cron)→베스트글 포인트

RLS: 공개글은 비로그인도 읽음 · 쓰기/수정/삭제는 본인 글만 · 공식 글·보상 지급은 service_role(서버)만.

Reward · 보상 경제(이사님 확정·farming-proof)

포인트는 '행동당'이 아니라 '선정으로만' 흐른다

왜 farming이 원천 차단되나
글·좋아요·해봤어요는 포인트가 0(좋아요/해봤어요는 '선정 신호'일 뿐). 포인트는 주간 베스트글 톱10·월간 대상이라는 사람이 본 결과로만 지급 → 양산·품앗이·자동화로 벌 수 있는 포인트가 없다. 포인트는 현금 환전 불가·우리 제품 전용이라 최악도 키트 마진과 상계.
구분지급비고
첫 글 온보딩+500P1회·멱등(콜드스타트 마중물)
주간 베스트글 1위+5,000P점수 = 좋아요×2 + 해봤어요×3
주간 2~3위+3,000P매주 월요일(KST) 정산
주간 4~10위+1,000P반응 0점 글은 제외(과지급 방지)
월간 대상+20,000P매월 1일(KST) 지난달 1위

공식 '코치' 글은 보상에서 제외(사람 글만). 사람 글이 쌓이고 반응이 붙어야 지급 → 초기엔 자연히 소액(월 ~8만 고정에 수렴).

Cold-start · 휑함 방지 + 자동 콘텐츠

UGC가 0이어도 꽉 차고, 매일 새 글이 올라온다

"초반엔 안 쓸 텐데 어떻게 휑함을 막나" — 우리가 이미 가진 데이터·큐레이션으로 채우고, 코치가 매일 노하우를 자동 게시한다.

① 코치 PICK 시드(상시)
인기 식재료에 큐레이션 노하우를 항상 표시 — 피드·도감 §6이 day 1부터 비지 않음.
② 월별 제철 챌린지
"이번 달 제철 OO 노하우 남기면 도감이 채워져요" — '쓸 거리'를 계획적으로 제공.
③ 자동 콘텐츠 드립 ⭐
'밀프레드 코치'가 매일 몇 개씩 새 노하우를 게시(신선한 타임스탬프·반응 가능). 콜드스타트와 '활발함'을 동시에.
④ '1호 마스터' 초대
아직 빈 식재료 = "첫 글 = 당신이 1호 마스터" 초대로 전환.

콘텐츠 생성 = AI 멀티에이전트 + 안전 검수

인기 식재료별로 영유아 편식 노하우를 대량 생성하고, 각 노하우를 적대 안전 검수(질식 형태·돌 전 꿀·알레르기·톤·현실성)로 교정해 풀에 적재. 드립 cron이 매일 풀에서 꺼내 게시(중복은 멱등 키로 차단).

정직성 원칙(astroturfing 금지)
가짜 '엄마' 계정으로 위장하지 않는다. 자동 게시 글은 작성자 '밀프레드 코치' · '🩺 코치 PICK' 배지로 명시 → 브랜드의 편집 콘텐츠임이 투명. 사람 노하우와 시각적으로 구분되고, 보상·랭킹에서도 제외.
Safety & Ops · 안전·운영

영유아 안전 가드 + 모더레이션

  • 위험 키워드 가드 — 글 작성 시 꿀(돌 전)·통포도/방울토마토·통 견과·생우유·통 떡 등 탐지 → 노란 안전 안내(코칭 안전 가드와 정합).
  • 자동 블라인드 — 신고 3건 누적 시 자동 숨김(올리브영식). 어드민에서 수동 숨기기/복구.
  • 톤 가드 — '비난·비교·강압·한 입만·판매 금지' 작성 안내. 자동 생성 콘텐츠도 동일 기준으로 검수.
  • 아동 개인정보 — 사진은 본인 촬영만·아이 얼굴 비공개 안내. UGC 라이선스·민감정보 조항은 /terms·/privacy에 반영됨.
