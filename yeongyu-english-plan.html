0. 한 줄 요약과 결정 요청 6건

이미 팔고 있는 요리키트 20종 위에 '영어 수업 레이어' 8종을 얹어, 영어유치원 정규 수업으로 파는 커리큘럼을 만든다. 키트 제조·배송·안전 설계는 현행 그대로 쓰므로, 신규 개발은 종이·인쇄물·영상 자료뿐이다.

1. 왜 지금인가 — 리서치 결론 5줄

  1. 전국 영유 820곳, 월 원비 135~154만원, 특별활동 규제 전무(법적으로 '학원'이라서) — 원장 단독 결정 + 지불력 최상의 B2B 채널. 우리 805 아웃리치 시트가 사실상 전수 리스트.
  2. 2026년 9월 레벨테스트(4세 고시) 금지 시행 → 올가을 설명회는 시험 어필이 불가능한 첫 모집 시즌. 업계 대응 가이드가 권장하는 전환 예시가 문자 그대로 "요리 놀이 중 'Can you pass me the carrot?'".
  3. 대형 8개 브랜드(폴리·i-Garten·메이플베어·PSA·SLP·애플트리·GDA·라이즈) 공식 커리큘럼에 쿠킹 정규 과목 전무, '영어요리' 전문 업체도 국내외 부재 — 확인된 화이트스페이스.
  4. 영유 원어민 교사 실태: "8년간 완성된 커리큘럼을 가진 학원을 한 곳도 못 봤다", 서울 영유 강사 63.9% 무자격 — 읽으면 되는 대본 + 봉투만 뜯는 키트가 원장에게는 품질 보증 장치.
  5. 영유 입학설명회 = 10월 초~11월 초에 이듬해 커리큘럼 발표 → 2027학년도 정규 편성 계약은 9월~10월 초 마감. 지금(7월)이 원장 미팅 적기. 놓치면 12~2월 겨울특강으로 중간 진입.

3. 개발 범위 — '영어 수업 레이어' 8종

기존 키트(재료+도구+사진 자산)는 그대로. 메뉴 1종당 아래를 신규 제작:

  1. Teacher's Guide(4p 내외) — 국제 표준 레슨플랜 구조(Objectives / Target Vocabulary / Materials / 3단 Procedure). 원어민용 영어 축어 대본에 tight(고정: 목표 문형·시연 순서·안전 수칙) / loose(재량: 게임·확장) 마킹 + 한국인 보조교사용 한국어 역할 지시 병기.
  2. 영어 레시피 포스터 — 기존 실사진에 영어 캡션(레벨별 문장 상한 적용), 코팅 인쇄.
  3. 재료 플래시카드 2크기(교사 제시용 / 게임용) — 기존 촬영 사진 재사용.
  4. 워크시트 3레벨 — L1(5세) 선긋기·색칠 / L2(6세) 단어+한 문장 / L3(7세) First-Then-Next 시퀀스문. 흑백 인쇄 겸용.
  5. 요리 챈트 1곡 + 제스처 시연 영상(1~2분) — 무자격 원어민도 따라 하는 최저 진입장벽 장치.
  6. 안전·알레르겐 시트 — 성분표 기반 한국형 알레르겐(우유·계란·밀·호두·메밀·잣 우선), 대체 구성 옵션, 가정 통지·서명 양식, 응급대응 카드.
  7. Daily Note(학부모 발송) — 오늘 배운 영어 문장 3개 + 사진 슬롯 + 가정 미션 1개("How does it taste?"). 원비 정당화 니즈 직격.
  8. 수용도×영어 체크리스트 — 만진/맛본 재료 × 사용 표현 2축. 코칭엔진 수용 5단계(거부→완식)와 동일 척도 → 추후 앱·리포트 연동.

프로그램 공통 자료(1회 제작): 원장용 제안서(한국어) + 커리큘럼 맵(영어·교수부장/원어민용) + 학부모 설명회 브로셔·슬라이드 + 연간·월간 교육계획안 양식 + 시연수업 킷.

레벨링: 대본은 1벌, 질문·기대응답만 3단 표기(국제 표준 differentiated outcomes — GSE pre-A1/A1/A2 = Cambridge Starters/Movers/Flyers = 5/6/7세). 예: L1 "Point to the flower!"(비언어 반응) / L2 "What colour is it?" → "Pink!" / L3 "What do we do first?" → "First, we press the dough." 요리 어휘가 Movers/Flyers(A2)급인 것은 "초등 영어학원 대비 어휘 선행" 세일즈 포인트로 사용.

7. 리서치 하이라이트 — 포지셔닝 근거

포지셔닝 한 문장

"영유가 유치원이 아니라서 잃는 것(오감·놀이·식습관)을, 영유가 가장 잘하는 방식(영어 몰입 정규수업)으로 채워주는 유일한 커리큘럼." 교사에겐 준비 0분 완성형, 학부모에겐 측정 가능한 아웃풋(어휘 카운트·수용도 리포트·레시피북), 원장에겐 규제 대응 + 리스크 흡수(알레르겐·위생 증빙).

이론 무장 — 전부 '영어 요리수업'을 지지

  • 몰입교육: 내용을 영어로 배우는 것이 몰입의 정의 그 자체 — 요리수업은 정통 몰입 콘텐츠.
  • Krashen: 실물 재료·시연 = 이해가능한 입력(i+1), 만들고 먹는 재미가 불안 필터 제거, 침묵기 아동도 동작으로 참여.
  • TPR: pour·mix·stir·taste = 명령문+신체동작 그 자체. 실물 키트는 '요리 흉내'가 아닌 진짜 다감각 연합.
  • CLIL: 요리가 대표 적용 교과로 명시된 국제 표준 접근 — "특별활동이 아니라 CLIL 정규수업"이라는 어휘 제공.
  • DAP·guided play: 학습식 영유의 최대 약점(발달적 부적합 비판)을 정규 영어수업 형태로 보완.

누리과정 격차 (영유가 못 하는 것)

  • 누리과정 신체운동·건강 영역에 "몸에 좋은 음식에 관심을 가지고 즐겁게 먹는다"가 국가 과정으로 명문화 — 유치원은 매일 하는 급식지도·요리활동·텃밭을 영유는 구조적으로 못 함(학원이라 적용 대상 아님).
  • 실증: 하루 착석 교습 ~5시간, 놀이터 전무, 영양사 없는 급식(전직 강사 증언·언론 보도). 학부모 선호 식생활 교육 1위 = 체험형 요리(51%).

식습관 × 영어의 학술 접합

  • 영국 교육부(DfE) 공식 가이드가 오감 식교육(Sensory food education)의 효익에 "언어 발달·어휘·자신감"을 명시 — "음식 오감 수업 = 언어 수업"을 영어권 정부 문서가 지지.
  • 반복 노출 근거: Wardle 2003 RCT + Nekitsing 2018 메타분석(n=4,017 — 효과가 노출 횟수에 비례) → 주 1회 × 학기 구독 모델 자체가 근거 기반.
  • 음식 테마 고유 이점: 배운 어휘를 당일 저녁 식탁에서 재사용 — 언어의 간격 반복과 음식의 반복 노출이 같은 루프에서 도는 유일한 테마.

8. 리스크 · 정직 가드

  • 법규 확인 2건(영업 병행): ① '영어 요리수업'의 교습과목 인정 범위(교육청 유권해석) ② 정기 납품 확대 시 식품 영업 형태 재확인.
  • 책임 구조: 밀프레드 = 생산물배상책임(PL)보험 필수 / 영유 = 학원 배상책임보험 확인 — 계약서에 책임분담·증권 교환·알레르겐 사전고지 조항.
  • 효과 주장 가드: "수업만으로 편식 치료" 금지(영국 Flavour School RCT — 프로그램 고유 효과 미입증). 정확한 주장 = "압박 없는 시식 맥락을 매주 설계 제공 + 가정 연계 미션으로 반복 노출 완성". 표본 절대수 대외 노출 금지. 식약처 푸드브릿지는 방법론 준거로만(정부 인증 뉘앙스 금지).
  • 알레르겐 한국형: 영유아 1~3위 = 우유·계란·밀, 호두·메밀·잣 경계 — 미국식 땅콩 중심 프로토콜 이식 금지.
  • 무료 경쟁자 차별선: 풀무원·CJ CSR 무료 식생활교육과 겹치지 않게 "정규 영어수업으로 기능하는 연간 커리큘럼"으로만 포지셔닝.
