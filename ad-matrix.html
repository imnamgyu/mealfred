📣 광고 변형 매트릭스 — A/B 풀

편식 부모 페인 8개 × 앵글 5개 = 40개 광고 변형. 메타·인스타 A/B용. 같은 페인이라도 앵글(공감·호기심·불안·또래증거·구체약속·즉시성·정체성)이 달라야 진짜 A/B가 됩니다. 클릭·가입 전환 1·2위 앵글의 기능을 첫 화면·온보딩에 올려 고도화.

페인 1 · 매 끼니가 전쟁 — 매일 감정 소모, 안 먹어 화나고 지침

P1-A공감/위로
오늘도 식판 앞에서 한숨 쉬셨다면, 엄마 잘못 아니에요.
매일 똑같은 전쟁에 지친 당신에게, 밀프레드가 오늘 하루치 응원 편지를 보내드려요.
🎬 비주얼 흰밥만 남긴 식판 옆, 식탁 모서리를 짚은 지친 엄마 손과 식어버린 국 한 그릇
🎯 타겟 3~5세 극단편식 아이를 키우며 매 끼니마다 자책하는 30대 전업맘
🧩 기능 매일 코칭 편지
P1-B호기심(질문형)
아이가 당근을 안 먹는 게, 사실 '맛' 때문이 아니라면요?
오늘의 질문 하나에 답하면, 우리 아이만의 안 먹는 진짜 이유가 보이기 시작해요.
🎬 비주얼 물음표가 떠 있는 말풍선과, 손도 안 댄 당근 한 조각을 빤히 쳐다보는 아이의 눈높이 컷
🎯 타겟 왜 안 먹는지 몰라 매번 같은 실랑이를 반복하는 워킹맘
🧩 기능 오늘의 질문
P1-C구체적약속(숫자·기간)
안 먹던 한 가지에서 시작해요. 좋아하는 음식에 살짝 얹는 것부터.
푸드브릿지가 아이가 이미 좋아하는 음식과 새 재료를 한 다리씩 연결해 줘요.
🎬 비주얼 좋아하는 계란밥 위에 잘게 다진 새 재료가 한 스푼 얹히는 클로즈업, 옆에 작은 화살표로 '한 다리씩'
🎯 타겟 새 재료만 보면 입을 꾹 다무는 4세 아이를 둔, 차근차근 가는 게 맞나 불안한 엄마
🧩 기능 푸드브릿지
P1-D사회적증거(또래·다수)
끼니마다 화내는 게 나만 그런 줄 알았어요.
같은 고민 하는 엄마들이 매일 작은 한 걸음씩, 밀프레드와 함께 가고 있어요.
🎬 비주얼 여러 가정의 식판 사진이 그리드로 모인 콜라주, 가운데 '오늘 한 걸음 함께한 엄마들' 문구
🎯 타겟 주변에 말 못 하고 혼자 끙끙 앓던, 또래 엄마들과 비교되어 위축된 첫째맘
🧩 기능 매일 코칭 편지
P1-E정체성(어떤 엄마)
억지로 먹이는 엄마 말고, 기다려 주는 엄마가 되고 싶었어요.
오늘 무엇을 시도하면 좋을지, 밀프레드가 매일 작은 질문 하나로 길을 안내해요.
🎬 비주얼 아이 옆에 무릎 맞대고 앉아 눈높이로 웃어 주는 엄마, 식탁엔 강요 없이 놓인 작은 접시 하나
🎯 타겟 혼내지 않고 키우고 싶지만 매 끼니 무너지는 자신에게 실망하는 가치지향형 엄마
🧩 기능 오늘의 질문

페인 2 · 새 음식을 주면 거부한다 — 시도→거부→좌절 반복

P2-A공감/위로
새 반찬 내밀 때마다 작아지는 건 아이가 아니라 나였어요
아이가 좋아하는 음식에 새 식재료를 살짝 얹어주는 '푸드브릿지'로, 거부 앞에서 더는 혼자 애쓰지 않게요.
🎬 비주얼 흰밥만 휑하게 남은 식판 옆, 숟가락을 든 채 멈춰 있는 엄마의 지친 손
🎯 타겟 3~5세 극단 편식 아이를 키우는, 매 끼니 거부에 자책하는 전업맘
🧩 기능 푸드브릿지(좋아하는 음식에 새 식재료를 다리 놓기)
P2-B호기심(질문형)
거부당한 그 음식, 정확히 언제 다시 꺼내야 할까요?
한 번 뱉었다고 끝이 아니에요. 아이마다 다른 '재노출 주기'를 코칭이 대신 알려드려요.
🎬 비주얼 냉장고 안에서 손도 안 댄 채 그대로 남은 브로콜리 한 봉지와 그 위에 떠 있는 물음표
🎯 타겟 '몇 번을 더 줘야 하나' 검색만 반복하는 첫째 키우는 초보맘
🧩 기능 거부 재노출 주기 코칭(다시 권할 타이밍 안내)
P2-C사회적증거(또래·다수)
어제 가지를 거부한 우리 아이, 또래들은 이렇게 다시 만났대요
거부 기록을 바탕으로 또래가 잘 받아들인 형태로 다리를 놓아주는 푸드브릿지, 같은 고민의 엄마들이 쓰고 있어요.
🎬 비주얼 비슷한 식판 사진이 격자로 나열되고, 어제는 빈칸이던 자리에 오늘은 작게 한 입 줄어든 반찬
🎯 타겟 내 아이만 유난인가 싶어 또래 비교가 궁금한 4~6세 외동맘
🧩 기능 푸드브릿지 + 또래가 잘 먹는 형태 안내
P2-D구체적약속(숫자·기간)
오늘 거부한 한 가지, 2주 동안 다리 놓는 법을 알려드려요
좋아하는 음식에 조금씩 얹는 푸드브릿지와 적절한 간격의 재노출, 그 작은 순서를 코칭이 짚어줘요.
🎬 비주얼 달력 위에 좋아하는 음식 사진에서 새 식재료 사진으로 점점 이어지는 작은 점선 다리
🎯 타겟 막연한 조언 말고 '뭘 언제'가 필요한, 계획형 워킹맘
🧩 기능 푸드브릿지 + 거부 재노출 주기 코칭(단계별 순서)
P2-E정체성(어떤 엄마)
억지로 떠먹이는 대신, 기다려주기로 한 엄마들이 있어요
강요 없이 좋아하는 음식에 다리를 놓고, 거부한 음식은 알맞은 때 다시 권하는 그 방식, 함께해요.
🎬 비주얼 한 입 거부한 아이를 다그치지 않고 그릇을 가만히 내려놓으며 미소 짓는 엄마의 옆모습
🎯 타겟 편식을 '훈육'이 아니라 '과정'으로 보고 싶은, 존중육아 지향 엄마
🧩 기능 강요 없는 푸드브릿지 + 재노출 주기 코칭

페인 3 · 뭘 먹여야 할지 모르겠다 — 영양 불안, 정답 없음

P3-A공감/위로
오늘도 흰밥만 남긴 식판 보면서, 내가 뭘 잘못한 걸까 싶었나요
엄마 탓 아니에요. 우리 아이 지금 식단이 어디가 비어있는지, 밀프레드가 조용히 같이 봐드릴게요.
🎬 비주얼 흰밥만 깨끗이 남고 반찬은 그대로인 식판 + 한숨 쉬며 정리하는 엄마의 손
🎯 타겟 3~5세 극단편식 아이를 둔, 매 끼니 죄책감을 느끼는 전업맘
🧩 기능 영양 신호등(31 KDRI 결핍 진단)
P3-B호기심(질문형)
매일 챙겨 먹이는데 우리 아이, 진짜로 부족한 영양소는 뭘까요?
오늘 먹인 식단표 사진 한 장이면, 어떤 영양소가 채워지고 비었는지 신호등 색으로 보여드려요.
🎬 비주얼 물음표가 떠 있는 평범한 집밥 식탁을 스마트폰 카메라로 비추는 장면
🎯 타겟 나름 신경 써서 차리지만 '이게 맞나' 늘 궁금한 6~7세 아이 엄마
🧩 기능 식단표 OCR(사진 한 장 진단)
P3-C사회적증거(또래·다수)
같은 또래 엄마들이 '뭘 먹여야 할지 모르겠다'에서 벗어난 방법
어림짐작 대신 31가지 영양소를 신호등으로 확인하는 엄마들이 늘고 있어요. 우리 아이 식단도 한번 비춰볼까요?
🎬 비주얼 여러 가정의 식판 썸네일이 신호등(초록·노랑) 배지와 함께 격자로 모인 화면
🎯 타겟 주변 엄마들 사이에서 정보 뒤처질까 신경 쓰는 4~6세 워킹맘
🧩 기능 영양 신호등(31 KDRI 결핍 진단)
P3-D즉시성/마법(사진 한 장)
어린이집 식단표, 사진만 찍어 올리면 영양 진단이 끝나요
메뉴 하나하나 적을 필요 없이 사진 한 장이면 OCR이 알아서 읽고, 부족한 영양소를 바로 짚어드려요.
🎬 비주얼 어린이집 월간 식단표를 폰으로 촬영하자 화면에 신호등 결과가 스르륵 채워지는 모션
🎯 타겟 기록할 시간조차 빠듯한, 등·하원 챙기는 맞벌이 엄마
🧩 기능 식단표 OCR(사진 한 장 진단)
P3-E정체성(어떤 엄마)
정답을 몰라도 괜찮아요, 우리 아이를 가장 잘 아는 건 엄마니까
감으로 불안해하지 않게, 31가지 영양소를 눈으로 확인하며 챙기는 엄마가 되어보세요.
🎬 비주얼 아이와 눈 맞추며 밥을 떠주는 엄마, 옆엔 신호등 진단이 떠 있는 폰 화면이 자연스럽게 놓인 모습
🎯 타겟 완벽하진 않아도 근거 있게 챙기고 싶은 5~7세 아이 엄마
🧩 기능 영양 신호등(31 KDRI 결핍 진단)

🎬 비주얼 1주차부터 4주차까지 흰밥 옆에 한 가지씩 색이 늘어나는 식판 4컷 타임라인

🎬 비주얼 숟가락을 들이밀지 않고 옆에서 함께 밥 먹는 엄마와 아이, 식판엔 흰밥과 작게 섞인 한 가지
🎯 타겟 혼내거나 협박하는 식사가 싫어 다른 길을 찾는 신념 있는 30대 엄마

페인 5 · 죄책감 — 내가 잘못 키우나, 우리 애만 이런가

오늘도 흰밥만 비운 식판 보면서, 내 탓 같으셨죠?
밀프레드 코칭 편지가 매일 한 통씩 "어머니 잘못이 아니에요, 이 시기 거부는 정상이에요"라고 먼저 말 걸어요.
🎬 비주얼 흰밥만 남고 반찬은 그대로인 식판 + 한숨 쉬며 식판 든 엄마의 손

🎬 비주얼 스마트폰 화면 속 '또래 비교' 그래프를 들여다보는 엄마의 옆모습, 표정이 조금 풀림

P5-C사회적증거(또래·다수)
사실 또래 열에 일곱은 새 음식 거부해요
밀프레드 또래 비교로 우리 아이가 평범한 범위 안에 있다는 걸 눈으로 확인해보세요.
🎬 비주얼 여러 아이의 식판 일러스트가 나란히, 대부분 비슷하게 반찬을 남긴 모습 한눈에
🎯 타겟 주변 엄마들과 비교하며 '우리 애만 늦나' 불안한 4세 첫째 엄마
🧩 기능 또래 비교 (같은 또래 분포 안에서 우리 아이 위치 확인)
P5-D정체성(어떤 엄마)
잘 먹이려 애쓰는 엄마일수록, 자책도 깊더라고요
그 마음을 아는 코칭 편지가 매일 옆에서 "충분히 잘하고 계세요"라고 다독여드려요.
🎬 비주얼 늦은 밤 식탁에 앉아 휴대폰 코칭 편지를 읽으며 옅게 미소 짓는 엄마, 따뜻한 조명
🎯 타겟 정성껏 준비한 반찬이 거부당할 때 자기 노력을 의심하는 완벽주의 성향 엄마
🧩 기능 코칭 편지 (애쓰는 부모를 탓하지 않고 다독이는 위로)
P5-E즉시성/마법(사진 한 장)
남긴 식판 사진 한 장이면, 자책 대신 답장이 와요
사진만 올리면 우리 아이 식습관을 또래와 비교해 읽어주고, 다음 날 위로의 코칭 편지가 도착해요.
🎬 비주얼 식판 사진을 찍어 올리자 화면에 또래 비교 결과와 코칭 편지 알림이 뜨는 휴대폰
🎯 타겟 기록할 여유 없이 바쁜, 사진 한 장이 최선인 둘째 키우는 워킹맘
🧩 기능 또래 비교 + 코칭 편지 (사진 한 장 → 비교 분석 → 다음 날 위로 편지)

페인 6 · 뭐가 부족한지·영양제 먹여야 하나 — 결핍 불안

P6-A공감/위로
오늘도 흰밥만 비우고 간 식판 앞에서, '내가 뭘 놓쳤나' 하셨죠
탓하려는 게 아니에요. 우리 아이 지금 식단에 뭐가 채워졌고 뭐가 비었는지, 영양 신호등이 색으로 가만히 보여드려요.
🎬 비주얼 흰밥만 남기고 반찬은 그대로인 어린이 식판 + 그 옆 지친 엄마 손이 살짝 받쳐 든 모습, 따뜻한 자연광
🎯 타겟 3~5세 극단 편식 아이를 둔, 매 끼니 죄책감에 시달리는 전업·워킹맘
🧩 기능 영양 신호등(31 KDRI) — 채워진 영양·비어 있는 영양을 색으로 한눈에
P6-B호기심(질문형)
영양제 사기 전에 — 진짜 부족한 게 뭔지 먼저 보고 사실래요?
기록한 끼니로 31가지 영양을 신호등처럼 보여주고, 빨간 칸엔 약 말고 '이 식재료 한 입'을 제안해드려요.
🎬 비주얼 한 손엔 알록달록 어린이 영양제 통, 다른 손엔 스마트폰 영양 신호등 화면 — 둘을 저울처럼 비교하는 구도
🎯 타겟 영양제 광고에 흔들려 장바구니에 담아두고도 '진짜 필요한가' 망설이는 6~8세 자녀 엄마
🧩 기능 결핍 영양소 → 보충 식재료 추천(약이 아닌 음식으로)
P6-C사회적증거(또래·다수)
비슷한 또래 엄마들도 '뭐가 부족한지 몰라서' 불안했대요
이제 막연한 걱정 대신 색으로 확인하세요. 31가지 영양 신호등이 빨강·노랑·초록으로 우리 아이 상태를 정리해줘요.
🎬 비주얼 여러 엄마의 휴대폰 화면이 모여 같은 영양 신호등 UI를 보고 있는 콜라주, 초록·노랑·빨강 칸이 또렷하게
🎯 타겟 맘카페에서 '우리 애만 편식인가' 검색해 본 적 있는 4~6세 자녀 엄마
🧩 기능 영양 신호등(31 KDRI) — 막연한 비교 대신 우리 아이 데이터 기반
P6-D구체적약속(숫자·기간)
31가지 영양, 빨간 칸 하나에 채워줄 식재료 한두 가지까지
부족한 영양마다 '이걸 더 먹여보세요' 식재료를 짚어드려요. 영양제 고민, 이번 주 한 끼부터 가볍게 바꿔보세요.
🎬 비주얼 31칸 영양 신호등 그리드에서 빨간 칸 하나를 손가락으로 누르자 '시금치·달걀' 같은 보충 식재료 카드가 펼쳐지는 화면
🎯 타겟 성장기 영양 결핍이 걱정돼 구체적인 '뭘 먹일지'를 원하는 5~7세 자녀 워킹맘
🧩 기능 결핍→보충 식재료 추천 + 31 KDRI 신호등
P6-E정체성(어떤 엄마)
불안에 영양제부터 사는 대신, 부족한 칸부터 들여다보는 엄마
추측 말고 데이터로. 우리 아이 끼니를 31가지 영양 신호등으로 보고, 빈 칸은 식재료 한 입으로 채워가요.
🎬 비주얼 냉장고 앞에서 휴대폰 영양 신호등을 보며 차분하게 장볼 거리를 적는 엄마, 옆엔 아이 그림이 붙은 화이트보드
🎯 타겟 감정적 불안보다 근거를 신뢰하는, 식습관을 길게 보고 가꾸려는 4~7세 자녀 엄마
🧩 기능 영양 신호등(31 KDRI) → 결핍 칸 보충 식재료로 잇는 흐름

페인 7 · 또래보다 작다·안 큰다 — 성장 불안

P7-A공감/위로
키 재고 돌아서면, 또 마음이 무거워지는 밤
혼자 걱정만 키우지 마세요. 우리 아이 키·몸무게가 또래 사이 어디쯤인지, 백분위 곡선으로 차분히 같이 봐드려요.
🎬 비주얼 잠든 아이 머리맡, 키 잰 흔적이 남은 벽지 앞에서 휴대폰을 든 엄마의 옆모습 (따뜻한 무드등 조명)
🎯 타겟 3~6세 아이가 또래보다 작아 단톡방·검진 때마다 속이 내려앉는, 자책 많은 첫째 엄마
🧩 기능 체위 시계열 그래프 + 또래 백분위 곡선(WHO LMS 기반)
P7-B호기심(질문형)
우리 아이, 또래 100명 중 몇 번째쯤일까요?
막연한 '작은 것 같아' 대신 숫자로. 키·몸무게 백분위와 BMI로 지금 위치를 한눈에 확인해보세요.
🎬 비주얼 아이들 100명이 작은 점으로 늘어선 백분위 곡선 위, 우리 아이 한 명만 또렷하게 표시된 깔끔한 인포그래픽
🎯 타겟 검진 결과지의 '하위 ○%' 숫자가 무슨 뜻인지 늘 헷갈렸던, 데이터로 납득하고 싶은 30대 엄마
🧩 기능 또래 비교 백분위 + BMI percentile 산출
P7-C사회적증거(또래·다수)
같은 또래 엄마들은, 키를 '느낌' 말고 '곡선'으로 봐요
단톡방 평균치 말고 우리 아이 곡선으로. 또래 분포 안에서 지금 어디인지, 추세는 올라가는지 같이 봐드려요.
🎬 비주얼 여러 엄마들의 휴대폰 화면에 같은 성장 곡선 앱이 떠 있는 플랫레이 (손만 보이는 구도)
🎯 타겟 주변 엄마들이 쓰는 건 일단 깔아보는, 또래 정보에 민감한 어린이집·놀이학교 학부모
🧩 기능 또래 분포 대비 위치 + 성장 추세선
P7-D구체적약속(숫자·기간)
키 1cm가 아니라, 곡선의 '방향'을 봅니다
한 번 재고 끝이 아니라 매달 기록이 쌓여 추세가 됩니다. 같은 백분위를 따라가는지, 성장에 필요한 영양은 채워지는지까지 함께요.
🎬 비주얼 월별로 점이 찍히며 우상향하는 성장 곡선 + 옆에 성장기 핵심 영양소 충족 체크리스트 화면
🎯 타겟 '클 때 되면 큰다'는 말로는 안 놓이는, 꾸준한 기록과 근거를 원하는 워킹맘
🧩 기능 월별 체위 시계열 추세 + 성장기 KDRI 영양 신호등
P7-E정체성(어떤 엄마)
조급해하기보다, 곁에서 지켜봐 주는 엄마이고 싶어서
닦달 대신 기록으로. 아이 성장을 또래 곡선 위에서 차분히 따라가며, 필요한 영양만 살며시 챙기는 방법.
🎬 비주얼 아이와 눈높이 맞춰 앉아 같이 식판을 보는 엄마, 강요 없이 편안한 식탁 분위기 (자연광)
🎯 타겟 비교·압박은 싫지만 우리 아이 성장만큼은 놓치고 싶지 않은, 존중 육아를 지향하는 엄마
🧩 기능 또래 비교 백분위 + 성장 영양 코칭(강요 없는 기록 기반)

페인 8 · 어린이집선 잘 먹는데 집선 안 먹는다

P8-A공감/위로
어린이집 식판은 싹 비워 오는데, 집밥은 왜 한 숟갈도 안 먹을까요
엄마 탓이 아니에요. 집밥과 기관밥을 따로 들여다보면, 우리 아이만의 패턴이 보여요.
🎬 비주얼 흰밥만 덩그러니 남은 집 식탁 + 옆에 깨끗이 비운 어린이집 식판 사진 두 장 나란히, 지친 엄마의 손
🎯 타겟 3~5세 아이가 집에서만 안 먹어 매 끼니 전쟁 치르는, 자책 많은 엄마
🧩 기능 장소별(집/기관) 분리 코칭 — 같은 아이의 집·기관 식사를 따로 분석
P8-B호기심(질문형)
같은 아이인데, 어린이집에선 잘 먹고 집에선 안 먹는 이유 궁금하지 않으세요?
어린이집 식단표 한 장만 찍으면, 집과 기관에서 뭐가 다른지 자동으로 보여줘요.
🎬 비주얼 물음표가 떠 있는 아이 식판 위로, 한쪽은 어린이집·한쪽은 집 장면이 갈라지는 스플릿 이미지
🎯 타겟 왜 집에서만 편식인지 원인을 알고 싶은, 정보 탐색형 4세 아이 엄마
🧩 기능 식단표 OCR — 기관 식단표 촬영 한 번으로 기관 식사 자동 기록
P8-C사회적증거(또래·다수)
"우리 애도 어린이집에선 잘 먹어요" — 생각보다 정말 흔한 일이에요
같은 고민을 가진 엄마들이 집·기관을 나눠 보며 우리 아이 패턴을 찾고 있어요.
🎬 비주얼 여러 가정의 집밥 식판 그리드가 모여 하나의 모자이크를 이루는 컷, 가운데 또래 아이들 실루엣
🎯 타겟 또래 다른 집은 어떤지 비교하며 안심하고 싶은 5세 아이 엄마
🧩 기능 장소별(집/기관) 분리 코칭 — 또래 흔한 패턴과 함께 우리 아이 위치 확인
P8-D구체적약속(숫자·기간)
식단표 한 장 찍어두면, 집에서 더 챙길 음식이 매일 한눈에 정리돼요
기관에서 먹은 건 OCR로 자동 채우고, 집에서만 비는 부분을 골라 알려드려요.
🎬 비주얼 달력 위에 어린이집 식단표 사진이 자동으로 채워지고, 집밥 칸에 '오늘 챙길 것' 한 줄이 뜨는 화면
🎯 타겟 매일 뭘 더 먹여야 할지 막막한, 효율을 중시하는 워킹맘
🧩 기능 식단표 OCR + 장소별 분리 코칭 — 기관 자동 기록 후 집에서 보충할 것만 제안
P8-E즉시성/마법(3초·사진 한 장)
어린이집 식단표, 그냥 사진 한 장만 찍어 올려보세요
메뉴를 일일이 적을 필요 없이, 찍는 순간 기관 식사가 기록되고 집밥과 비교돼요.
🎬 비주얼 엄마 손이 냉장고에 붙은 식단표를 폰으로 찰칵 — 화면에 메뉴들이 자동으로 채워지는 모션 컷
🎯 타겟 기록할 시간도 의지도 없는, 손이 가벼운 게 제일 중요한 바쁜 엄마
🧩 기능 식단표 OCR — 촬영 한 번으로 기관 메뉴 자동 입력
✅ 실행 — 페인당 상위 2~3개 앵글을 동시 게재 → CTR·가입 CPA로 승자 추림 → 승자 앵글의 기능을 온보딩 전면으로. 이 전환 숫자가 곧 다음 라운드 '앱 수요 증거'(YC 과제와 동일). 비주얼 컨셉은 그대로 디자이너/생성AI 브리프로 사용.
